When a user looks at a search result, they expect to see a list of items to look into. If there are no results, don't give them noisy text which can be taken as a search result - the page should be clean.
 ![search-result-bad-list](/PublishingImages/search-result-bad-list)Figure: Bad example - The list of "suggestions" is just noise and can confuse the usermg src="/PublishingImages/search-result-bad-icon" alt="search-result-bad-icon" /&gt;Figure: Bad example - Having an icon implies that an error happened which is not the case![search-result-good-web](/PublishingImages/search-result-good-web)Figure: Good example - Plain and clean screen![search-result-good-iphone](/PublishingImages/search-result-good-iphone)Figure: Good example - Plain and clean screen on mobile
