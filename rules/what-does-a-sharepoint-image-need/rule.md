 As a minumum, the SharePoint image needs:  


1. Windows Server 2003 or 2008
2. SQL Server 2005 sp1
3. WSS 3.0 sp1
4. MOSS 2007 sp1
5. MS Office SharePoint Designer sp1
6. Visual Studio .NET 2005 or 2008
